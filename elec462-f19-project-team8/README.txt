# NewMC

    https://github.com/solocat/NewMC


## How to run


    ./NewMC
    
    KEY_UP, KEY_DOWN: 커서를 위/아래로 이동
    Insert: 커서 위치의 항목 선택
    Home, End: 현재 페이지의 맨 위/아래로 이동
    Pg_up, Pg_dn: 페이지를 앞/뒤로 이동
    Enter: 현재 커서 위치의 디렉토리로 이동
    F1: 도움말
    F2: 선택한 항목의 내용 보기
    F3: 현재 디렉토리에서 파일 또는 폴더 검색
    F4: 현재 디렉토리의 항목 전체선택/해제
    F5: 선택한 항목 복사
    F6: 선택한 항목 이동 또는 이름 변경
    F7: 디렉토리 생성
    F8: 선택한 항목 삭제
    F9: 프로그램 종료



## Environments for testing code


    This program can be run anywhere in your computer's directory.
